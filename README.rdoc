= redmine-wsd

Redmine wiki page macros for www.websequencediagrams.com

== Usage

  {{seq
    note over A,B: text1
    note left of A: text2
    note right of A
        multiline
        text
    end note
    A-->B: text
    loop text
        A->B: text
    end
  }}

=== Options

==== width

  {{seq(800px)
    note over A,B: text1
    :
  }}

==== style

  {{seq(800px, napkin)
    note over A,B: text1
    :
  }}
